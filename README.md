# IntelliCare-For-All
" IntelliCare For All " is a project designed to do Medical Diagnose using symptoms like fever, cough etc . The model is trained over open source dataset provided over kaggle .
The goal is to make **accessible, affordable, and intelligent healthcare support** available to everyone — especially in urban environments where both accessibility and time are critical.
AI-Powered Healthcare for Everyone

# 🎯 Features
* 🔍 Symptom-based disease prediction
* 🤖 Machine Learning model integration
* 📊 User-friendly interface (Streamlit-based)
* 💊 Suggests possible treatments (future scope)
* 🌍 Designed for inclusivity (usable by all sections of society)

# 🛠️ Tech Stack
Frontend: Streamlit
Backend: Python
Machine Learning: Scikit-learn
Libraries: Pandas, NumPy

## 🚀 How to Run the Project
```bash

# Clone the repository
git clone https://github.com/your-username/intellicare-for-all.git

# Navigate to project folder
cd intellicare-for-all

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```
# 🧠 How It Works
1. User chooses symptoms
2. Features are extracted
3. Trained ML model predicts disease
4. Result is displayed with suggestions
5. Near by Hospitals are also recommended
